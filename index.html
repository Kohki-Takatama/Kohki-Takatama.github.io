カスタマイズ項目の紹介

カスタムした項目

  • ビデオ・スクショ・トレース

    • on / off / 失敗のみ / リトライのみ

  • テストタイムアウトの長さ

  • テストするブラウザ

便利そうなもの

  • デバイス設定

    • Desktop Chrome / Pixel 5 / iPhone 12 など
    • "Project"という単位で、複数設定も可能

  • ベース URL の設定

    • /user/profile など path のみの記述ができる

まとめと展望

まとめ

  • Playwright という E2E ツールがある

  • ユーザー視点の自動テストができて便利

  • （視覚的な）記録を残せる

展望

  • エラー対応力

    • エラーメッセージ少(ターミナル)

  • テストの前後処理

  • CI（Github Actions など）

  • UI モード

    • codegen（自動生成）

  • テスト設計思想
